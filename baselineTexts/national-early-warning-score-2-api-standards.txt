Overview

Use this API standard to enable sharing of patients' vital signs and National Early Warning Score 2 (NEWS2) scores between systems and across care boundaries, for example, between two hospitals, or between GP and ambulance systems.

We have developed a number of API standards for different purposes which are outlined below.

Before you begin any development work using these API standards, contact us to discuss your best options.

API status
